# zoom/pan box

## setup
```
    yarn && yarn start
```

## Usage

Pass children into JavascriptZoom component like this: 
```
    <JavascriptZoom>
        <h1>Awesome content</h1>
        <Rectangle />
        <Circle />
        <TextArea>
            <p>This is a zoomable box</p>
        </TextArea>
        </Area>
    </JavascriptZoom>
```
